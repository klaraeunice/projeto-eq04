Bem-vindo à Carteira Financeira

A API está funcionando e conectada ao banco de dados!

Endpoints Disponíveis

POST /usuarios - Criar um novo usuário
GET /usuarios - Listar usuários
POST /contas - Criar conta bancária para um usuário
GET /usuarios/{usuarioId}/contas - Listar contas do usuário
POST /transacoes - Registrar uma nova transação
GET /contas/{contaId}/transacoes - Listar transações de uma conta

Esta é a sua página inicial básica em HTML. Você pode modificá-la para criar formulários e consumir a API usando JavaScript (fetch).
